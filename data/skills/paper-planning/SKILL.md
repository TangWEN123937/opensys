## 论文大纲拟定流程（本技能被激活时执行）

### 第一步：确定论文结构类型

根据研究主题和论文类型，选择合适的结构：

**实证研究型**（最常见）：
1. 摘要 → 2. 引言 → 3. 文献综述 → 4. 研究设计与方法 → 5. 数据分析与结果 → 6. 讨论 → 7. 结论与展望 → 参考文献

**理论研究型**：
1. 摘要 → 2. 引言 → 3. 理论基础 → 4. 理论框架构建 → 5. 理论分析与论证 → 6. 结论与展望 → 参考文献

**案例研究型**：
1. 摘要 → 2. 引言 → 3. 文献综述与理论框架 → 4. 案例介绍 → 5. 案例分析 → 6. 讨论与启示 → 7. 结论 → 参考文献

**学位论文扩展型**（硕博）：
在基础结构上增加：绪论（研究背景+意义+目标+方法+框架图）、独立的理论基础章、更详细的研究方法论证

### 第二步：拟定详细大纲

为每个章节标注：

```markdown
## 论文大纲

### 第一章 绪论（约 X 字，占比 10-15%）
- 1.1 研究背景：[要点]
- 1.2 研究问题与目的：[要点]
- 1.3 研究意义：理论意义 + 实践意义
- 1.4 研究方法与技术路线：[要点]
- 1.5 论文结构安排

### 第二章 文献综述（约 X 字，占比 20-25%）
- 2.1 核心概念界定：[需要界定的概念]
- 2.2 国内研究现状：[主要流派和观点]
- 2.3 国外研究现状：[主要流派和观点]
- 2.4 研究评述与空白：[现有研究不足 + 本文切入点]
  - 论据来源：[引用哪些文献]

### 第三章 理论基础与研究框架（约 X 字，占比 10-15%）
- 3.1 理论基础：[采用的理论]
- 3.2 分析框架：[构建的分析框架]
- 3.3 研究假设（如有）：[假设列表]

### 第四章 研究设计与方法（约 X 字，占比 10-15%）
- 4.1 研究方法选择：[定性/定量/混合，为什么]
- 4.2 数据来源与样本：[数据来源]
- 4.3 分析工具与过程：[分析方法]

### 第五章 分析与讨论（约 X 字，占比 25-30%）
- 5.1 [分析维度1]：[核心发现]
- 5.2 [分析维度2]：[核心发现]
- 5.3 综合讨论：[发现的意义和解释]

### 第六章 结论与展望（约 X 字，占比 5-10%）
- 6.1 主要结论
- 6.2 理论贡献与实践启示
- 6.3 研究局限
- 6.4 未来研究方向

### 参考文献
### 致谢（学位论文）
### 附录（如有）
```

### 第三步：生成摘要与关键词初稿

1. **中文摘要**（200-300 字）：研究背景→研究方法→主要发现→结论
2. **英文摘要**（Abstract）：对应中文摘要的英文翻译
3. **关键词**：3-5 个，中英文对照

### 第四步：用户确认

使用 `ask_user` 展示完整大纲，请用户：
- 确认论文结构和章节安排
- 确认各章节字数分配是否合理
- 补充遗漏的要点或调整重点
- 确认研究方法和分析框架

### 注意事项

- **参考前序产出物**：必须基于 Phase 1 的需求确认单和 Phase 4 的文献综述成果
- **字数分配合理**：文献综述和分析讨论是重点章节，合计占 50% 以上
- **逻辑闭环**：引言提出的问题 → 综述论证必要性 → 方法说明可行性 → 分析回答问题 → 结论总结回答
- **学位论文注意**：需要比期刊论文更详细的绪论和方法论证
