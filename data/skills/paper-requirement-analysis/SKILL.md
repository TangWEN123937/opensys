## 论文需求分析流程（本技能被激活时执行）

在开始任何学术论文写作之前，必须先完成以下需求确认。使用 `ask_user` 工具逐项确认不明确的参数。

### 第一步：明确论文基本参数

逐项确认以下信息（如用户未提供，必须用 ask_user 询问）：

1. **研究主题与问题**：
   - 核心研究主题（如"城市更新中的社区参与机制"）
   - 具体研究问题或假设（如果已有）
   - 研究范围和边界（时间/地域/对象限定）
2. **论文类型**：
   - 期刊论文（目标期刊名称和级别：SCI/CSSCI/核心/普刊）
   - 学位论文（本科/硕士/博士，学校名称）
   - 会议论文（目标会议名称）
3. **格式规范**：
   - 引用格式：GB/T 7714（国标）/ APA / MLA / Chicago / 学校自定义格式
   - 是否有模板文件（学校或期刊提供的 Word/LaTeX 模板）
   - 特殊格式要求（如摘要中英文双语、关键词数量等）
4. **字数要求**：
   - 总字数范围（如 8000-10000 字）
   - 各章节字数建议分配（如文献综述不超过 30%）
5. **时间节点**：截止日期、中间检查节点
6. **参考资料**：
   - 用户是否已有相关文献（PDF 文件路径）
   - 是否有导师推荐的必读文献
   - 是否有已完成的开题报告或前期研究

### 第二步：分析研究定位

基于确认的参数，输出以下分析：

1. **研究定位**：一句话概括本论文要解决什么学术问题
2. **潜在创新点方向**：
   - 理论创新（新框架/新模型/新概念）
   - 方法创新（新的研究方法或分析工具）
   - 视角创新（跨学科/新角度/新对象）
3. **预计文献检索关键词**：中英文各 3-5 个核心检索词，供后续文献搜索阶段使用
4. **预计论文结构**：基于论文类型给出初步结构建议

### 第三步：输出需求确认单

用以下格式整理确认结果，供后续阶段参考：

```
## 论文需求确认单

- 研究主题：[核心主题]
- 研究问题：[具体研究问题或假设]
- 论文类型：[期刊/学位/会议] — [具体目标]
- 字数要求：[目标字数范围]
- 引用格式：[GB/T 7714 / APA / MLA / 其他]
- 格式模板：[有/无，路径或说明]
- 截止日期：[日期]
- 已有资料：[文献路径/开题报告等]
- 研究定位：[一句话定位]
- 创新点方向：[潜在创新点]
- 检索关键词：
  - 中文：[关键词1, 关键词2, ...]
  - 英文：[keyword1, keyword2, ...]
- 预计结构：[章节列表]
```

### 注意事项

- **不要假设参数**：学术写作格式规范差异巨大，必须明确确认
- **区分论文类型**：期刊论文和学位论文在结构、字数、深度要求上有本质差异
- **检索关键词要精准**：直接影响后续文献搜索的质量，必须和用户确认
- **简洁提问**：一次 ask_user 可以包含多个问题，避免反复打断用户
